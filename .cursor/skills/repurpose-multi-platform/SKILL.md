## What this Skill does (and doesn’t)
把你的“原始素材”（文章/笔记/脚本/要点）做结构化提炼，然后按平台规则**重新组织**（而不是机械改写），保持一致的人设与核心观点。

不做的事：
- 不在没有证据时把内容写得像“权威报告”；会标注“需补证据”。
- 不做违规强引流；会给各平台更安全的引导写法。

## Progressive disclosure
常驻规则只保留“抽取主张→平台翻译→一致性检查”。更长的品牌词表/禁用词/语气卡放在：
- `references/`：brand-voice.md、do-not-say.md、platform-length.md 等（你可后续补齐）。

## Core instruction (always-on)
### 0) 最小输入
若用户没说清，先问/抽取：
- **Source**：原始素材（粘贴/要点/链接摘要）
- **Platforms**：要改写成哪些平台（至少 2 个）
- **Audience**：目标人群
- **Goal**：收藏/关注/私信/咨询/成交/背书
- **Voice**：人设/品牌语气（专业/松弛/犀利/克制/故事）
- **Constraints**：不能说（功效、价格、引流、竞品等）

### 1) 先做“内容核”抽取（必须）
先把源内容压成一个“内容核”，用于一致性：
- **One-liner thesis**：一句主张（≤ 25 字）
- **3 proof points**：三个证据点（案例/数据/流程/对比）
- **1 story**：一个故事/场景（可选）
- **CTA intent**：希望用户做的动作（收藏/评论/私信等）

### 2) 平台翻译规则（按平台重组，不做同义替换）
生成时遵循：
- **小红书**：开头先给收益/结论；正文偏“步骤/清单/避坑”；更像笔记；可加“我怎么做”的过程
- **抖音/短视频口播**：短句口语；先 Hook；主体 3 点；结尾 CTA；给字幕友好版本
- **公众号**：标题可情绪但承接要稳；“铺垫→论证→案例→总结”；段落清晰，信息密度更高
- **LinkedIn**：观点先行；少修辞；“立场→证据→方法→提问”；更职业
- **英文跨境短视频**：避免中式直译；动作动词 + 场景；更直接的 benefit；短句

### 3) 输出规格（默认）
对每个平台输出一套：
- **Title options (5)**：5 个标题
- **Opening (3)**：3 个开头（或口播 Hook）
- **Body**：正文/脚本（平台适配后的结构）
- **CTA (2)**：保守版与转化版各 1
- **Hashtags/keywords**（可选）：3-8 个

### 4) 一致性检查（必须）
发布前做一致性校验：
- 核心主张是否一致（不互相打架）
- 证据点是否被夸大（跨平台更易夸张）
- 人设语气是否一致（不要一边“导师口吻”一边“姐妹体”）
- CTA 是否与平台习惯一致（LinkedIn 不要太硬广）

## Usage examples (copy/paste)
### Example 1：小红书 ↔ 公众号
**用户输入：**
这是我写的公众号草稿（粘贴）。请改成小红书图文笔记版：更像真实分享、分步骤、可收藏。保留核心观点但不要书面。再给 5 个小红书标题。

### Example 2：同主题多平台（含 LinkedIn）
**用户输入：**
我做 B2B 市场，这段内容想一稿多发到：LinkedIn + 公众号 + 抖音口播。语气要专业克制，不要夸大承诺。请分别输出每个平台版本，并确保核心观点一致：……（粘贴要点）

### Example 3：中文 → 英文 TikTok
**用户输入：**
把这段中文内容改成 TikTok 英文口播：更直接、更口语、更短句。给 2 个不同开头（一个稳健，一个反常识），并给字幕版本：……（粘贴原文）

## Troubleshooting
### 1) “改完像复制粘贴，平台感不强”
我会：
- 先重做“内容核”，再按平台重排结构
- 强制改写开头与段落节奏（而不是替换同义词）
- 为每个平台加入其常见“承接方式”（XHS 步骤、LinkedIn 观点+提问）

### 2) “人设不一致/风格跑偏”
我会：
- 让你用 3 句话定义人设语气（你也可以直接给 3 条旧内容当样本）
- 建立“用词黑名单/偏好句式”
- 对每个平台做语气对齐复审

### 3) “担心夸大/违规”
我会：
- 输出“安全版”与“更强转化版”两套 CTA
- 把绝对化/保证类词汇替换为条件化表达
- 标注需要证据的句子（未提供证据则降级表述）

## Quality checklist (before publishing)
- [ ] **先有内容核**：一句主张 + 3 证据点清晰
- [ ] **结构真适配**：不是同一段话换几处词
- [ ] **人设一致**：语气、称呼、口头禅一致
- [ ] **承诺可兑现**：不夸大效果，不虚构背书
- [ ] **CTA合平台**：平台友好，不硬引流
- [ ] **可直接发布**：标题/开头/正文/CTA齐全
