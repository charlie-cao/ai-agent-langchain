# ⚖️ Project 11 — 法律合规分析 Agent

## 商业价值说明书

### 为什么这个 Agent 有市场需求？
中国法律服务市场规模超 3000 亿，但 95% 的中小企业负担不起专业法律顾问（律师时费 500-5000 元/小时）。核心痛点：
- **合同审核贵且慢**：律所审合同至少 3000 元起，3-5 个工作日
- **法规变化快**：劳动法、数据合规（PIPL）、知识产权等频繁更新
- **风险识别困难**：非法律专业的创业者看不懂合同陷阱

本 Agent 将法律文件分析成本从 3000 元降为 0，将时间从 3 天缩短为 3 分钟。

## 项目简介

基于 LangChain + Ollama 的法律合规 Agent，支持合同条款分析、风险识别、法规检索和合规建议生成。

## 技术架构

```
用户上传合同/法律问题
    ↓
contract_analyzer.py → 条款提取 + 风险标注 (高/中/低风险)
law_retriever.py     → 法规 BM25 检索 (劳动法/合同法/数据法)
compliance_checker.py → PIPL/等保/行业规范合规检查
    ↓
agent.py             → 综合分析报告生成
    ↓
app.py (Streamlit)   ← UI
api.py (FastAPI+SSE) ← REST API
```

## 功能特性

- ✅ 合同条款风险评级（高/中/低）
- ✅ 法规条文检索（劳动法、合同法、数据保护法）
- ✅ 数据合规检查（PIPL、网络安全法）
- ✅ 合规报告自动生成
- ✅ 流式 AI 法律问答
