# docs/changelog.md — 客服 Agent 变更日志

## v1.0.0（首次发布）

### 新增
- ReAct Agent，基础工具11个，qwen3.5:latest 模型支持
- 知识库工具：BM25全文搜索，12篇中文FAQ
- 情感分析工具：关键词权重评分，自动升级判断
- 意图分类工具：7类意图规则匹配
- 工单系统：创建/查询/更新/升级，P1-P4优先级+SLA
- 订单查询工具：4个模拟订单，包含完整物流轨迹
- Streamlit UI：4模式（智能客服/订单查询/工单管理/知识库）
- FastAPI SSE接口，端口8008
- 用户级别多轮会话记忆（最近10轮）
- 23个单元测试，全部通过
