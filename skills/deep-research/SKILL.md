# Deep Research

深度调研技能。

## 使用场景

- 专题研究（行业分析、竞品调研、技术评估）
- 事件深度调查（背景、现状、影响、未来预测）
- 人物/公司背景调查
- 市场调研报告

## 工作流程

```
第1步：明确调研目标
    ↓ 确定范围、深度、时间跨度
第2步：信息采集
    ↓ 多个信源交叉采集
第3步：初步整理
    ↓ 按维度分类、去重、验证
第4步：深度挖掘
    ↓ 针对薄弱环节定向补充
第5步：结构化输出
    ↓ 形成报告
第6步：事实核查
    ↓ 标注信息来源、标记存疑点
```

## 采集策略

### 信息金字塔

```
         ┌──────────┐
         │  专家/一手 │  ← 原始数据、行业报告、官方发布
         ├──────────┤
         │  新闻/媒体 │  ← 多媒体报道、交叉验证
         ├──────────┤
         │  聚合/二手 │  ← 综述、评论、分析
         └──────────┘
```

### 信源分类

| 类型 | 示例 | 权重 |
|------|------|------|
| 官方发布 | 政府网站、公司公告 | ⭐⭐⭐⭐⭐ |
| 一手数据 | 财报、论文、专利 | ⭐⭐⭐⭐⭐ |
| 权威媒体 | BBC、Reuters、财新 | ⭐⭐⭐⭐ |
| 行业媒体 | 36kr、虎嗅、TechCrunch | ⭐⭐⭐ |
| 社交讨论 | Twitter、Reddit、微博 | ⭐⭐ |

```markdown

[3句话核心结论]

[历史沿革、现状概览]

## 二、关键事实
### 2.1 [维度1]
- 要点1（[来源]）
- 要点2（[来源]）

### 2.2 [维度2]
...

## 三、关键数据
| 指标 | 数值 | 来源 |
|------|------|------|
| ... | ... | ... |

## 四、各方观点
### 支持方
...
### 反对方
...
### 中立分析
...

## 五、未来预测
- 短期（6个月内）：...
- 中期（1-3年）：...
- 长期（3年+）：...

## 六、风险与不确定因素
⚠️ [存疑点] - [原因]

## 七、参考来源
1. [标题](URL)
2. ...
```

## 核查清单

- [ ] 同一事实至少2个独立信源确认
- [ ] 数据标注来源和采集时间
- [ ] 观点和事实分开表述
- [ ] 明确标注"未经证实"的内容
- [ ] 利益相关方视角需注明

## 注意事项

- 不要预设立场，先采集再分析
- 警惕单一来源的"独家爆料"
- 数据要注明统计口径和时间范围
- 预测要说明假设前提
