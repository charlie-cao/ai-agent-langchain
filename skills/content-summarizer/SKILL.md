# Content Summarizer

内容摘要与改写技能。

## 使用场景

- 对长文章进行压缩摘要
- 将视频/音频内容转文字并摘要
- 多篇内容合并为一份综述
- 将内容改写为不同风格/长度

## 工具

### 1. summarize CLI（推荐）
```bash
# 摘要URL或文件
npx summarize "URL"
npx summarize "file.pdf"
npx summarize "video.mp4"
```

### 2. Gemini 深度摘要
```bash
node C:/Users/charlie/.openclaw/workspace/skills/gemini/SKILL.md # 查看详细用法
```

### 3. 本地 Ollama（可选）
```bash
ollama run llama3 "请总结以下内容：[粘贴内容]"
```

## 摘要风格

| 场景 | 长度 | 风格 |
|------|------|------|
| 新闻快讯 | 3-5句 | 简洁、客观、事实为主 |
| 深度分析 | 200-500字 | 结构化，含观点 |
| 社媒帖子 | 推文格式 | 抓眼球、有悬念 |
| 内部报告 | 完整段落 | 详细、含数据 |

## 输出格式

```
## 📝 摘要：[来源标题]

### 一句话
[最核心的信息，20字内]

### 要点
- 要点1
- 要点2
- 要点3

### 关键引用
> "[金句或关键数据]"

### 原文链接
[URL]
```

## 注意事项

- 先确认内容质量，低质量内容可直接标记"无价值"
- 技术文档保留方法/步骤/参数
- 新闻类保留时间线和关键人物
- 视频摘要优先提取关键帧时间戳
