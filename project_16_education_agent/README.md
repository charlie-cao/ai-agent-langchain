# 🎓 Project 16 — 智能教育 Agent

## 商业价值说明书

### 为什么这个 Agent 有市场需求？
中国 K12 教育市场规模超 1 万亿，在线教育 2023 年重新復苏。核心痛点：
- **学习计划千人一面**：传统课程无法针对每个学生的薄弱点定制
- **练习题质量参差不齐**：好的习题库需要大量人工标注
- **学习进度难追踪**：家长和老师缺乏实时数据支撑决策
- **自学效率低**：没有及时反馈，学生练习白费

本 Agent 实现个性化自适应学习：根据学习目标和薄弱点生成定制化课程表，自动生成多类型测验，实时批改并追踪进度。

### 市场竞品分析
| 竞品 | 优势 | 劣势 |
|------|------|------|
| 作业帮/猿辅导 | 用户体量大 | 题库封闭，AI辅助弱 |
| Khan Academy | 自适应好 | 英文为主 |
| Duolingo | 游戏化强 | 仅语言学习 |
| **本项目** | 本地运行，可定制学科，免费 | 需接入题库内容 |

## 项目简介

自适应学习 AI Agent，支持个性化学习计划生成、多题型测验（选择/填空/判断）、自动批改和进度追踪。

## 技术架构

```
学生输入 (目标/薄弱点/学习时间)
    ↓
study_planner.py → 自适应学习计划
                    按每周可用小时分配模块
                    薄弱点优先安排
    ↓
                → 多题型测验生成:
                    multiple_choice / fill_blank / true_false
                → 答案批改 (精确匹配 + 关键词匹配)
                → 进度追踪 (历史存储)
    ↓
agent.py        → 综合调度
    ↓
app.py (Streamlit) ← UI
api.py (FastAPI)   ← REST API
```

## 功能特性

- ✅ 自适应学习计划（按目标+薄弱点定制）
- ✅ 3种测验题型（选择题/填空题/判断题）
- ✅ 自动批改（支持关键词部分匹配）
- ✅ 学习进度追踪（历史记录）
- ✅ 每周学时智能分配
