# 🛒 Project 18 — 电商智能 Agent

## 商业价值说明书

### 为什么这个 Agent 有市场需求？
中国电商市场规模超 13 万亿，竞争极为激烈。商家面临的核心痛点：
- **竞品价格监控耗时**：手动对比竞品价格需要每天花 2-3 小时
- **定价策略滞后**：价格调整跟不上市场变化，利润流失
- **商品描述质量差**：商家手写描述缺乏吸引力，转化率低
- **信息不对称**：不知道竞争对手什么时候降价、促销

本 Agent 实现全天候竞品价格监控 + 自动定价建议 + AI 商品描述生成，帮助商家保持竞争力。

### 市场竞品分析
| 竞品 | 优势 | 劣势 |
|------|------|------|
| 扫货达人 | 功能垂直 | 仅价格监控，无AI |
| 万里牛 | ERP集成好 | 价格高，学习曲线陡 |
| 自建爬虫 | 灵活 | 维护成本高，易被封 |
| **本项目** | AI增强，本地运行，免费 | 演示数据，需接入实际爬虫 |

## 项目简介

电商智能 AI Agent，集成竞品价格监控、智能定价建议（基于市场均价和配置阈值）、AI 商品描述生成（突出卖点/情感化/简洁版三种风格）。

## 技术架构

```
输入: 商品信息 / 竞品链接列表
    ↓
price_monitor.py → 竞品价格抓取 (mock数据演示)
                   市场均价/最低价/最高价计算
                   价格超阈值告警 (默认 5% 偏差)
                   定价建议 (低于/高于/持平均价)
    ↓
                → AI商品描述生成:
                   highlight (强调卖点)
                   emotional (情感化叙述)
                   minimal (简洁版)
    ↓
agent.py        → 综合调度
    ↓
app.py (Streamlit) ← UI
api.py (FastAPI)   ← REST API
```

## 功能特性

- ✅ 竞品价格实时对比（支持多竞品）
- ✅ 价格偏差告警（可配置阈值 %）
- ✅ 3种定价建议（低价/相当/高价 + 行动建议）
- ✅ AI 商品描述生成（3种风格）
- ✅ 市场价格统计（均价/最低/最高）
