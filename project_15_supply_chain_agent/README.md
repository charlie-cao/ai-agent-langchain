# 🚚 Project 15 — 供应链优化 Agent

## 商业价值说明书

### 为什么这个 Agent 有市场需求？
中国制造业供应链管理市场规模超 2 万亿，库存积压和缺货造成的损失每年超千亿。核心痛点：
- **库存臃肿**：过量采购导致资金占用，仓储成本高
- **缺货损失**：需求预测不准导致断货，丢失订单
- **配送路线低效**：多仓/多门店配送路线未优化，物流成本虚高
- **人工预测不准**：销售季节性、促销影响难以人工建模

本 Agent 通过移动平均需求预测 + 安全库存计算 + TSP 路径优化，实现库存水位自动化管理。

### 市场竞品分析
| 竞品 | 优势 | 劣势 |
|------|------|------|
| SAP 供应链云 | 功能最全 | 极贵，实施 6-12 个月 |

## 项目简介

供应链优化 AI Agent，集成库存管理（移动平均需求预测、安全库存、补货建议）和路径规划（最近邻 TSP 启发式算法）。

## 技术架构

```
输入: 库存数据 / 配送节点列表
    ↓
inventory_tool.py  → 库存天数计算 (stock / daily_demand)
                     低库存警报 (<7天)
                     补货量建议 (安全库存+缓冲)
                     移动平均需求预测 (滑动窗口)
    ↓
route_optimizer.py → 最近邻 TSP 启发式
                     总距离/时间计算
                     停靠点排序优化
    ↓
agent.py           → 综合调度
    ↓
app.py (Streamlit) ← UI
api.py (FastAPI)   ← REST API
```

## 功能特性

- ✅ 库存健康度评估（充足/临界/缺货）
- ✅ 3期移动平均需求预测
- ✅ 自动补货建议（含安全库存缓冲）
- ✅ TSP 路径优化（最近邻启发式）
- ✅ 配送时间估算（距离/速度模型）
